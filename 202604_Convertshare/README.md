# Welcome to ConvertShare

ConvertShare is an experimental tool for exploring the **cost-effectiveness of going all‑electric in residential properties in New York City**. The goal is to help users quickly understand how different property locations and characteristics might impact costs and savings.

Our application lets you:
- View properties and analyze them by different radius-based groupings.
- Get a high-level breakdown and rough cost/savings estimations for each area.
- Generate AI-powered summaries to make the data easier to understand.

## AI Summaries & Disclaimer

ConvertShare uses AI to generate summaries and insights. AI systems can **hallucinate** or produce outputs that are incomplete, inaccurate, or outdated. Nothing in this application should be treated as professional, financial, legal, or engineering advice.

You should:
- Treat all AI-generated content as a starting point, not a final answer.
- Verify and fact-check results with qualified professionals, domain experts, and/or trusted tools.
- Use this project as one input among many when making real-world decisions.
